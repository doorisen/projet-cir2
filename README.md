# Pour l'initialisation du site :
- dans le terminal linux, lancer apache2 avec ``sudo service apache2 start``
- dans le terminal linx, lancer mariadb avec ``sudo service mariadb start``

- dans le navigateur, aller sur http://projet-cir2-39/phpmyadmin
- se connecter avec : identifiant : ``irve``, mot de passe : ``irve``
- cliquer sur la table ``irve_bretagne`` si elle existe
- dans l'onglet ``Importer``, cliquer sur ``choisir un fichier`` et choisir le fichier ``/var/www/html/projet-cir2/config/irve_bretagne.sql``, et enfin cliquer sur ``Importer`` en bas de la page



# Pour accéder au site :
- accéder au site sur http://projet-cir2-39/projet-cir2
- si c'est la première visite du site après initialisation, un écran de chargement (gif de pikachu) s'affichera le temps que la database soit automatiquement peuplée
